# Day 29 学习笔记：Docker 基础

## 1. 核心概念理解
- **容器与虚拟机的区别**：容器共享宿主机内核，更轻量、启动更快；虚拟机拥有独立内核，更安全但笨重。
- **Docker 架构**：Client (命令行) -> Daemon (后台进程) -> Images (只读模板) -> Containers (运行实例)。

## 2. 今日实验记录
- **Hello World**：成功运行并验证了 Docker 引擎。
- **Ubuntu 交互模式**：进入容器内部，确认了容器拥有独立的文件系统。
- **Nginx 服务**：通过 `-p 8080:80` 将容器服务暴露到了宿主机。

## 3. 常用指令总结
- `docker run`：下载并运行镜像。
- `docker ps`：查看正在运行的容器。
- `docker stop`：停止容器。
- `docker rm`：删除容器。
